R101 - An Introduction to R

Course characteristic

This course has three main goals. The first is to reduce the degree of dependence (ideally to eliminate it) from paid statistical software, which may be unavailable for financial reasons and which provides the user with relatively limited work options.

Secondly, this course aims to promote interest in R as the programming language with (almost) endless analytical possibilities and a vibrant global community.

Last but not least, I would like you to have fun. Learning something like doing data analysis in R may not be easy. However, it could be quite enjoyable too. I admire Andy Field for his humorous, yet comprehensive way of teaching complex abstract topics such as factor rotation. My humble aspiration is to make learning R joyful, even if for a bit.

So, what exactly can you expect?

Covered topics:

  1. PROGRAMMING LANGUAGE R
  1. Conceptual introduction to R
  2. Installing R and RStudio, an introduction to the environment, libraries
  3. Fundamentals – using R as a calculator, assigning values to objects, objects classes
  4. Coding etiquette
